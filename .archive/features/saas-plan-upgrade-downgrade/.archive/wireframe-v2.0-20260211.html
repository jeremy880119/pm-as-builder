W6: D1 勸退 Modal
S3-001
⚠

確定要降級嗎？

從 Pro 降至 Free 後，你將失去以下功能：

  • 失去進階數據分析
  • 失去下載數據報表
  • 自動幫所有 12 集插入廣告
  • 總計 8 條單集 Flink 萬用連結失效
  • 每月失去 5 集 AI 內容萃取
  • 每筆經營會員抽成漲價 3%

帳戶資訊 Card (元件 A) Card + Badge + Button + Alert --shadow-sm, --border, --radius-md Click "變更方案" → navigate to /plans. "取消降級" → PATCH cancel schedule. Full-width on mobile, max-width 600px on desktop
方案選擇頁 (元件 B) Toggle + Card grid + Button + Badge --primary, --border, --radius-md Toggle switches billing cycle, prices re-render. Card CTA: upgrade → Stripe Checkout, downgrade → D1 Modal. 4-col → 2-col (tablet) → 1-col (mobile). Toggle always visible.
升級成功頁 (元件 E) Empty State (success variant) --radius-md, --shadow-sm "立即體驗" → navigate to dashboard or newly unlocked feature. Centered, max-width 520px, padded on mobile
Paywall 鎖頭 (Sidebar) Nav item + Lock Icon + Badge --radius-full (lock circle), --muted bg (badge) Click locked item → open Paywall Modal (元件 C). Sidebar collapses to bottom tab / hamburger on mobile
Paywall Modal (元件 C) Modal (480px) + Button + Badge --overlay, --shadow-xl, --radius-lg CTA → Stripe Checkout or /plans. Close X → dismiss. 480px → full-width with 16px padding on mobile
D1 勸退 Modal Modal (640px) + Icon + List + Button pair --overlay, --shadow-xl, --radius-lg "取消" → dismiss modal. "仍要降級" → hard boundary check API. 640px → full-width with 16px padding on mobile
D2 攔截 Modal Modal (640px) + Card items + Button pair --overlay, --shadow-xl, --radius-lg "保留目前方案" → dismiss. "我已截圖" → dismiss + log event. 640px → full-width with 16px padding on mobile
D3 Error Banner Alert/Banner (full-width, top) --error bg, white text Not closable. "查看詳情" → open D3 Modal. Full-width, text wraps on mobile
D3 Modal Modal (640px) + Card items + Button --overlay, --shadow-xl, --radius-lg "聯繫客服" → open support channel / Intercom. 640px → full-width with 16px padding on mobile
