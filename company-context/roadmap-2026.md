# 2026 年度產品發展藍圖

本文件涵蓋廣告系統的「精準」投放與 SaaS 平台的「整合」功能兩大主軸。

---

## 1. Ads「精準」：投放、配速與系統重構

廣告系統在 2026 年的重點在於提升投放的精確度、自動化監控以及系統的穩定性。

### 季度規劃

| 季度 | 重點項目 | 說明 |
|------|----------|------|
| **Q1** | 精準投放與配速 | 鎖定地區、時段與裝置；廣告競品研究、工程估時；測試「完聽率」可能性 |
| **Q2** | 三大系統重構 | 自動週報生成、系統穩定性檢查與調整、建立完善監控系統 |
| **Q3** | 進階功能導入 | 導入 auto balance 以提升預測準確度 |
| **Q4** | 手動觸發支援 | 支援關鍵操作的手動觸發（如從 approve 退回或下架廣告） |

### 關鍵指標 (Key Inputs)

| 指標類型 | 項目 |
|----------|------|
| Efficiency1 | 人工處理時間 |
| Efficiency2 | 廣告填充率 |
| Breadth1 | 進案數 |
| Depth | 廣告單價 |

## 2. SaaS「整合」：團隊管理、影音與數據

SaaS 的發展核心在於解決銷售痛點，透過「Killing Features」吸引大咖客戶轉移平台，並優化數據回收。

### 季度規劃

| 季度 | 重點項目 | 說明 |
|------|----------|------|
| **Q1** | 影音與數據整合 | 多平台數據整合（Spotify API）；Video Podcast 上傳整合（YouTube API 雛形）；數據回收價值評估 |
| **Q2** | 權限與架構改動 | Team management（權限管理）；底層架構改動：單一使用者驗證轉向「Show 下的多使用者」管理模式；整合 Stripe 加 seat 付費功能 |
| **Q3** | 數據回收強化 | 多平台數據回收（Apple 爬蟲、YouTube API） |
| **Q4** | UX 優化 | 升降級 UX 調整；單集頁面排序功能 |

### 關鍵指標 (Key Inputs)

| 指標類型 | 項目 |
|----------|------|
| Breadth1 | MAU |
| Depth | ARPPU |
| Efficiency | 轉化率 |
| Frequency | 留存率 |
