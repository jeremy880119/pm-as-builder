流程說明

升級入口

  • 入口 A： 帳戶頁「升級」按鈕 → 方案選擇頁 → 結帳頁
  • 入口 B： 直接進入方案選擇頁（from marketing）
  • 入口 C： 功能鎖頭觸發 Paywall Modal → 升級或查看方案

降級流程分支

  • 主動降級： 用戶發起 → 勸退 → 原因收集 → 當下檢核 → 完成
  • 被動降級： 付款失敗 → 自動降級 → Email 通知 + Banner 提示

試用機制

  • 首次訂閱： 7 天免費試用，結帳頁顯示試用 badge
  • 試用中： 帳戶頁顯示剩餘天數與進度條
  • 試用結束： 自動降級至 Free，顯示升級 CTA
  • 重複訂閱： 無試用期，結帳頁註明「立即收費」
