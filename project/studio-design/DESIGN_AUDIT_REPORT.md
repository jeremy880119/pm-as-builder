## 一、色彩系統 (Color Palette)

### 1.1 現有色彩盤點

| 用途 | 色彩值（推估） | 使用位置 |
|------|---------------|----------|
| **品牌主色** | `#F06A6A` / `#E85A5A` | 側邊欄 CTA、主要按鈕、連結 |
| **成功/綠色** | `#4ADE80` / `#22C55E` | 公告橫幅背景、成功狀態 |
| **資訊/藍色** | `#60A5FA` / `#3B82F6` | 引導橫幅背景、圖表、連結文字 |
| **警告/黃色** | `#FCD34D` | 部分提示（較少使用）|
| **背景灰** | `#F9FAFB` / `#F3F4F6` | 頁面背景、表格斑馬紋 |
| **邊框灰** | `#E5E7EB` / `#D1D5DB` | 卡片邊框、分隔線 |
| **文字深灰** | `#1F2937` / `#374151` | 標題、主要文字 |
| **文字淺灰** | `#6B7280` / `#9CA3AF` | 次要文字、說明文字 |
| **漸層** | `#FFE4E6` → `#DBEAFE` | Dashboard 卡片背景 |

### 1.2 色彩問題

| 問題 | 嚴重度 | 說明 |
|------|--------|------|
| ⚠️ 語意不一致 | 高 | 綠色同時用於「成功公告」和「試用中/進行中」狀態，語意混淆 |
| ⚠️ 主色變體不明 | 中 | 品牌粉紅色的 hover、active、disabled 狀態未標準化 |
| ⚠️ 連結色混用 | 中 | 有時用粉紅色、有時用藍色作為連結 |
| ⚠️ 圖表色彩 | 低 | 圖表使用青色 (#2DD4BF)，與整體粉紅調性不協調 |

## 二、字體排版 (Typography)

### 2.1 現有字體盤點

| 層級 | 估計樣式 | 使用位置 |
|------|----------|----------|
| **頁面大標** | 20-24px, font-weight: 600 | 「成效分析」「會員名單」等 |
| **區塊標題** | 16-18px, font-weight: 600 | 卡片標題、section 標題 |
| **正文** | 14px, font-weight: 400 | 段落、說明文字 |
| **標籤/輔助** | 12-13px, font-weight: 400 | 表頭、tag、次要資訊 |
| **數字強調** | 24-32px, font-weight: 700 | Dashboard 統計數字 |

### 2.2 字體問題

| 問題 | 嚴重度 | 說明 |
|------|--------|------|
| ⚠️ 字重不一致 | 中 | 相同層級標題有時 500、有時 600 |
| ⚠️ 行高不統一 | 中 | 部分長段落行高過窄，閱讀吃力 |
| ⚠️ 數字字體 | 低 | 統計數字建議使用 tabular-nums 對齊 |

## 三、間距系統 (Spacing)

### 3.1 現有間距模式

| 位置 | 估計值 | 備註 |
|------|--------|------|
| **頁面左右 padding** | 24-32px | 大致一致 |
| **區塊間距 (section gap)** | 24-32px | 不一致 |
| **卡片內 padding** | 16-24px | 混用 |
| **表格行高** | 48-56px | 不一致 |
| **按鈕內 padding** | 8-12px / 16-24px | 混用 |

### 3.2 間距問題

| 問題 | 嚴重度 | 說明 |
|------|--------|------|
| ⚠️ 無 spacing scale | 高 | 缺乏 4px/8px 基準的間距系統 |
| ⚠️ 區塊間距不一 | 中 | 有些區塊緊密、有些鬆散 |
| ⚠️ 表格密度不一 | 中 | 不同表格的行高、padding 不同 |

## 四、組件盤點 (Component Inventory)

### 4.1 按鈕 (Buttons)

| 類型 | 樣式 | 使用位置 | 問題 |
|------|------|----------|------|
| **Primary** | 粉紅實心 + 白字 + 圓角 | 「上傳單集」「儲存」「新增」「確認」 | 圓角不一（有 4px、8px、full） |
| **Secondary** | 白底 + 粉紅框 + 粉紅字 | 「檢視」「編輯」| 有時無框只有文字 |
| **Text/Link** | 無背景 + 粉紅/藍字 | 「看教學」「點擊了解」| 顏色混用 |
| **Icon Button** | 圓形/方形 + icon | 編輯、刪除、more | 大小不一 |

**按鈕問題：**
- 圓角半徑不統一（4px / 8px / 16px / full）
- 按鈕高度不一（32px / 36px / 40px）
- 部分按鈕缺少 hover 狀態視覺回饋

### 4.2 卡片 (Cards)

| 類型 | 樣式 | 使用位置 |
|------|------|----------|
| **統計卡片** | 白底 + 淺陰影 + 漸層背景 | Dashboard |
| **內容卡片** | 白底 + 1px 邊框 | 設定頁面 |
| **列表卡片** | 白底 + 無邊框 | 第三方串接 |
| **警示卡片** | 淺黃/淺粉背景 + 左邊線 | 提示訊息 |

**卡片問題：**
- 陰影深度不一（有些有、有些無）
- 圓角半徑混用（8px / 12px / 16px）
- 背景處理方式不一致

### 4.3 表格 (Tables)

| 元素 | 現況 | 問題 |
|------|------|------|
| **表頭** | 灰底、12-14px | 有些有底色、有些無 |
| **表格行** | 白底 + hover 淺灰 | hover 效果不一致 |
| **Checkbox** | 粉紅色 checked | 大致一致 |
| **分頁** | `< > 1-20` 樣式 | 一致 |

### 4.4 Tab 導航 (Tabs)

| 類型 | 樣式 | 使用位置 | 問題 |
|------|------|----------|------|
| **下劃線式** | active 下方粉紅線 | 成效分析、多平台推廣 | ✅ 主流 |
| **按鈕式** | active 粉紅背景 | 聽眾回饋（留言/Apple/語音） | ⚠️ 混用 |
| **Pills** | 圓角標籤式 | 單集列表（已發佈/草稿） | ⚠️ 混用 |

### 4.5 表單元素 (Form Elements)

| 元素 | 樣式 | 問題 |
|------|------|------|
| **Input** | 白底 + 灰框 + 8px 圓角 | 圓角不一 |
| **Toggle** | 粉紅色 on / 灰色 off | ✅ 一致 |
| **Select/Dropdown** | 白底 + 灰框 | 下拉樣式待確認 |
| **Rich Text Editor** | 工具列 + 白底 | ✅ 一致 |
| **Radio/Checkbox** | 粉紅色 checked | ✅ 一致 |

### 4.6 警示與通知 (Alerts & Banners)

| 類型 | 樣式 | 使用位置 |
|------|------|----------|
| **成功 Banner** | 綠底 + 深綠字 + ✓ icon | 頂部推廣公告 |
| **資訊 Banner** | 藍底 + 深藍字 + ℹ icon | 設定引導公告 |
| **行內提示** | 淺黃底 + 左橙線 | 表格上方說明 |
| **行內警示** | 淺粉底 + 左紅線 | 開啟廣告設定 |

### 4.7 Modal 彈窗

| 觀察 | 說明 |
|------|------|
| **背景** | 半透明黑色遮罩 |
| **卡片** | 白底 + 16px 圓角 + 陰影 |
| **關閉** | 右上角 X |
| **CTA** | 底部置中粉紅按鈕 |

### 4.8 Stepper 步驟指示

| 觀察 | 說明 |
|------|------|
| **樣式** | 數字圓圈 + 連接線 |
| **Active** | 粉紅實心圓 + 白字 |
| **Complete** | 粉紅勾勾 |
| **Upcoming** | 灰框 + 灰字 |

### 4.9 空狀態 (Empty States)

| 頁面 | 處理方式 | 問題 |
|------|----------|------|
| Dashboard（升級） | 有插圖 + 說明 + CTA | ✅ 好 |
| 表格無資料 | 只有空白 | ⚠️ 缺乏引導 |
| 聽眾回饋 | 有說明文字 | ⚠️ 缺插圖 |

## 五、重大不一致問題摘要

### 🔴 高優先級

| # | 問題 | 影響範圍 | 建議 |
|---|------|----------|------|
| 1 | **按鈕圓角不統一** | 全站 | 統一為 8px 或 6px |
| 2 | **Tab 導航樣式混用** | 5+ 頁面 | 統一為下劃線式 |
| 3 | **狀態顏色語意混亂** | 表格、標籤 | 定義語意色彩系統 |
| 4 | **缺乏 spacing scale** | 全站 | 建立 4px 基準系統 |

### 🟡 中優先級

| # | 問題 | 影響範圍 | 建議 |
|---|------|----------|------|
| 5 | 卡片陰影/邊框混用 | Dashboard、設定 | 統一卡片樣式 |
| 6 | 連結顏色不一（粉/藍） | 全站 | 統一為品牌色 |
| 7 | 表格行高不一 | 列表頁 | 統一為 52px |
| 8 | 按鈕高度不一 | 全站 | 定義 sm/md/lg 尺寸 |

### 🟢 低優先級

| # | 問題 | 影響範圍 | 建議 |
|---|------|----------|------|
| 9 | 空狀態處理不一 | 部分頁面 | 建立空狀態模板 |
| 10 | 數字未使用等寬字型 | 統計數字 | 加入 tabular-nums |

## 六、現有設計優點

值得保留並標準化的設計模式：

| 優點 | 說明 |
|------|------|
| ✅ **品牌色鮮明** | 珊瑚粉作為主色，辨識度高、符合友善調性 |

| ✅ **Stepper 設計** | 步驟流程視覺清晰 |
| ✅ **Checkbox 樣式** | 品牌色 checked 狀態一致 |

## 七、設計系統建立建議

### Phase 1：Design Tokens（基礎）
1. 定義色彩系統（Primary、Secondary、Semantic、Neutral）
2. 定義字體階層（H1-H6、Body、Caption、Label）
3. 定義間距系統（4px 基準：4, 8, 12, 16, 24, 32, 48, 64）
4. 定義圓角系統（sm: 4px, md: 8px, lg: 12px, full）
5. 定義陰影系統（sm, md, lg）

### Phase 2：Core Components（核心組件）
1. Button（Primary, Secondary, Ghost, Icon）
2. Card（統一邊框 or 陰影）
3. Table（標準化表頭、行高、hover）
4. Tabs（統一為下劃線式）
5. Input / Select / Toggle
6. Alert / Banner
7. Modal
8. Empty State

### Phase 3：Patterns（組合模式）
1. Page Header（標題 + 說明 + 操作按鈕）
2. Data Table（搜尋 + 過濾 + 表格 + 分頁）
3. Settings Section（標題 + 說明 + 表單）
4. Wizard/Stepper（步驟流程）
